## How to Use

1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/RanitManik/javascript-projects.git
   ```

2. Navigate to the specific challenge directory.
   ```bash
   cd javascript-projects/challengeX
   ```

## Contributions
